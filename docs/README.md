# TypoFix — документація

TypoFix — нативний кросплатформний (Windows + macOS) застосунок, що автоматично
визначає, коли текст набрано в неправильній розкладці клавіатури, перемикає
розкладку і **перенабирає** вже введені «крякозябри» у правильній мові.
Сучасний, приватний (нуль телеметрії), легкий аналог Punto Switcher із
першочерговою якісною підтримкою **української** та **англійської** мов.

## Документи (читати в цьому порядку)

1. **[CONCEPT.md](./CONCEPT.md)** — ідея, проблема, цільові користувачі,
   принципи, чим перевершуємо Punto. *Точка входу для будь-якого нового AI/розробника.*
2. **[ARCHITECTURE.md](./ARCHITECTURE.md)** — технічна архітектура: Rust+Tauri,
   модулі, платформена абстракція, двигун розпізнавання, формати даних, потоки.
3. **[DEVELOPMENT_PLAN.md](./DEVELOPMENT_PLAN.md)** — повний поетапний план
   розробки з чеклістами і критеріями готовності (Definition of Done).
4. **[TESTING_STRATEGY.md](./TESTING_STRATEGY.md)** — стратегія тестування й
   **автономної перевірки**: детермінований core, віртуальна платформа,
   golden-фікстури, replay-харнес. Ключове для самостійного пошуку й виправлення багів.

## Статус

🟡 Планування завершено. Реалізація ще не почата.

## Швидкі факти

- **Стек:** Rust (ядро + платформа) + Tauri (трей + вікно налаштувань).
- **Цільові ОС:** Windows (розробка/перша ціль), macOS (повноцінний порт). Linux — закладено архітектурно.
- **Мови тексту:** uk + en у першу чергу; фундамент для будь-яких інших.
- **Принцип:** усе локально, нуль телеметрії, малий слід у пам'яті, миттєвий hot-path.
