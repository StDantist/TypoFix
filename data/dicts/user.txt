# Особистий словник користувача — «НІКОЛИ НЕ ЧІПАТИ»-набір (veto).
# ============================================================================
# Сюди дописуй слова/терміни, які TypoFix НЕ повинен перемикати в іншу
# розкладку. Це твій персональний whitelist поза стандартним словником:
# жаргон, нікнейми, бренди, абревіатури, рідкісні слова, які надто нечасті,
# щоб потрапити в авто-словник (напр. сленг), але які ти набираєш свідомо.
#
# Формат:
#   * один термін на рядок;
#   * `#` на початку рядка — коментар (ігнорується);
#   * порожні рядки ігноруються;
#   * регістр НЕ важливий (матчинг регістронезалежний);
#   * пробіли по краях рядка обрізаються.
#
# Як це працює: loader (typofix_data::load_user_words) читає цей файл у рантаймі;
# core тримає набір як veto — якщо набране слово збігається з рядком звідси,
# розкладка НЕ перемикається. Зміни підхоплюються при перезавантаженні даних
# (перезапуск/reload), без перезбірки.
#
# Приклади (розкоментуй або заміни своїми):
# лох
# крякозябри
# нік
# EURUSD
# scalping
#
# --- Твої слова нижче ---
