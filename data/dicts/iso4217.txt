# ISO 4217 — активні коди валют (alphabetic codes), один код на рядок, UPPERCASE.
# Джерело: стандарт ISO 4217 (переліки кодів — факти/стандарт, не об'єкт копірайту).
# Призначення: дані для core (Den). Veto-правило (у core, НЕ тут): токен
#   ^[A-Z]{6}$, де [0:3] ∈ ISO ∩ [3:6] ∈ ISO  →  валютна пара (EURUSD, GBPUSD,
#   XAUUSD…) → НЕ перемикати розкладку. Системно: список кодів + патерн, не хардкод пар.
# Формат: один код на рядок, UPPERCASE, `#` — коментар, порожні рядки ігноруються.
# Loader: typofix_data::load_iso4217(&Path) -> HashSet<String> (parse_iso4217 для рядка).
#
# СВІДОМО ВИКЛЮЧЕНО (зменшує хибні veto на 6-літерних UPPERCASE-токенах):
#   - тестові/службові: XXX (no currency), XTS (testing);
#   - bond market units: XBA XBB XBC XBD; supranational accounting: XUA XSU;
#   - fund-коди (не торгуються як готівка): BOV CHE CHW CLF COU MXV USN UYI UYW.
# Дорогоцінні метали (XAU/XAG/XPT/XPD) ЗАЛИШЕНО — це реальні Forex-інструменти
